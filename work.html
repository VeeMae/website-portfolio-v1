The Meet Application

The Meet App is a serverless, progressive web application. It was built using React, a Google Calendar API to fetch upcoming events, serverless functions through AWS lambda, OAuth2 authentication flow, and a test-driven development technique. Through the app, users are able to filter events by city, show/hide event details, specify a number of events to be displayed, view cached events when offline, add a shortcut to their device homescreen, and view a chart showing the number of events by city and popularity. In addition, the process included creating user stories to define the application logic and using Atatus as an application performance monitoring tool post-production.

MyFlix Application (React Version)

I built this application using the MERN Stack. It was created in 2 phases - 1) server-side using REST API and MongoDB 2) client-side using React and Redux. The app allows users to register and log in, view movies from a database, update/delete their profile, and add/remove movies to/from their list of favorites.

Technical Specs:

  • ⚙️ API uses Node and Express
  • ⚙️ API uses REST architecture
  • ⚙️ API uses middleware modules
  • ⚙️ Database is built using MongoDB
  • ⚙️ Business logic is modeled with Mongoose
  • ⚙️ Coded in JavaScript
  • ⚙️ Uses React library
  • ⚙️ API is tested using Postman
  • ⚙️ API includes user authentication and authorization
  • ⚙️ API is deployed to Heroku
  • ⚙️ Programmed in Visual Studio Code
