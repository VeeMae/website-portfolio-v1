The Meet Application

The Meet App is a serverless, progressive web application. It was built using React, a Google Calendar API to fetch upcoming events, serverless functions through AWS lambda, OAuth2 authentication flow, and a test-driven development technique. Through the app, users are able to filter events by city, show/hide event details, specify a number of events to be displayed, view cached events when offline, add a shortcut to their device homescreen, and view a chart showing the number of events by city and popularity. In addition, the process included creating user stories to define the application logic and using Atatus as an application performance monitoring tool post-production.
